<h1>CMS Wiki</h1>
<p>CMS Wiki - это беспланый движок для вики сайта, созданный на базе Laravel и базы данных MySQL.</p>
<h2 class="mt-4">Особенности установки:</h2> 
<ul>
<li>1. Обязательна регистрация главного пользователя под логином admin.</li>
<li>2. На старте нужно импортировать дамп базы данных, расположенного в каталоге /public, файл называется CMSwiki.sql</li>
<li>3. Также нужно прописать данные для подключения к базе данных в файле .env</li>
</ul>
 <h2 class="mt-4">Возможности редактора</h2>
  <ul>
    <li>Поддерживаются все стандартные <code>HTML</code>-теги: 
      <code>h1</code>, <code>p</code>, <code>a</code>, <code>ul</code>, <code>img</code> и другие.</li>
    <li>Доступно оформление с помощью <strong>Bootstrap-классов</strong> — просто добавляйте, например:
      <code>class="btn btn-primary"</code> или <code>class="alert alert-info"</code>.</li>
    <li>Можно использовать таблицы, списки, цитаты, блоки кода и любые другие HTML-элементы.</li>
  </ul>

  <div class="alert alert-info mt-4">
    <strong>Совет:</strong> оформляйте материал с помощью заголовков, списков и ссылок — так страницы будут выглядеть аккуратно и структурировано.
  </div>

  <h2 class="mt-4">Добавление изображений</h2>
  <p>Чтобы вставить картинку в текст:</p>
  <ol>
    <li>Загрузите файл в папку <code>/public/img</code> вашего проекта.</li>
    <li>Добавьте HTML-тег с путём к изображению. Например:</li>
  </ol>

  <pre><code>img src="/img/example.jpg" alt="Описание изображения" class="img-fluid rounded"</code></pre>

  <p class="mt-3">
    После сохранения изменений изображение появится прямо на странице.
  </p>

  <h2 class="mt-4">Создание новых страниц</h2>
  <p>
    Чтобы создать новую страницу в CMS Wiki, достаточно разместить в тексте ссылку вида:<br>
    <code>< ahref="/index/new" >Новая страница< / a></code>
  </p>
  <p>
    После перехода по этой ссылке система автоматически создаст страницу с указанным именем.  
    Обратите внимание: уровень <strong>index</strong> является обязательным в структуре URL.
  </p>

  <hr>

  <h2 class="mt-4">Советы по оформлению</h2>
  <ul>
    <li>Используйте заголовки <code>h2</code> и <code>h3</code> для структуры текста.</li>
    <li>Применяйте классы Bootstrap для красивого оформления без CSS.</li>
    <li>Добавляйте изображения, видео и ссылки для более насыщенного контента.</li>
  </ul>
